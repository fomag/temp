Help with—One-time passcode

In order to validate your identity, the CRA sent a one-time passcode to the telephone number that you provided when you enrolled for multi-factor authentication.

If you have selected text message the code will be sent via SMS. If you have selected a telephone call, the code will be verbally provided to you in an automated message.

Please enter your one-time passcode once you receive it.

If the telephone number displayed is not correct, please select a different phone number or contact us for help.

Once the code has been accepted, you can manage your multi-factor authentication settings including add/change telephone numbers.

Help with—Resend Passcode

Selecting resend passcode will send a different one-time passcode to the telephone number you have already chosen. This may take a few minutes. You will be locked out if you try too many times.
